ilo.statistics Installation
---------------------------

To install ilo.statistics using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``ilo.statistics`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        ilo.statistics

* Re-run buildout, e.g. with:

    $ ./bin/buildout
